# 财务报表分析 Agent 标准工作流程 (Analysis Framework)

> 基于张新民、钱爱民《财务报表分析（第6版·立体化数字教材版）》
> 核心方法论：传统比率分析法 + 项目质量分析法，定性分析补充定量分析，共性分析与个性分析协调统一

---

## 一、教材分析路径总览

教材第8章明确提出综合分析路径：

```
资本结构质量 → 资产质量 → 利润质量 → 现金流量质量
     ↑              ↑           ↑            ↑
  资本引入战略   资源配置战略  战略实施后果  现金支撑力度
```

逻辑链条：资本引入战略决定资本结构 → 资本结构影响控制权和治理结构 → 决定资源配置战略 → 资源配置战略通过资产结构和质量体现 → 利润质量揭示战略实施后果 → 现金流量质量体现利润质量和战略实施的现金支撑。

---

## 二、分析总体目标

1. **评价企业财务状况质量**：资产质量、资本结构质量、利润质量、现金流量质量四个维度
2. **评估企业经营业绩**：包括盈利水平、盈利能力和盈利可持续性
3. **预测企业未来发展前景**：基于趋势、战略布局和资源禀赋
4. **为经济决策提供信息支持**：满足投资者、债权人、管理层、监管机构等多方需求

---

## 三、Agent 自动化工作流程（11步）

### 第 1 步：背景分析与数据输入

- **目标**：了解公司基本情况、行业背景和治理结构，加载财务数据
- **输入**：年报PDF / 财务数据 / 股票代码
- **处理动作**：
  1. 提取公司基本信息（证券代码、名称、行业、主营业务）
  2. 了解控股股东及实际控制人情况（判断治理结构）
  3. 了解行业竞争态势和公司战略定位（来自MD&A）
  4. 提取四表一注结构化数据
  5. 校验数据完整性
- **输出**：公司背景摘要 + 结构化财务数据集 + 数据完整性报告
- **工具**：`read_financial_report`, `annual_report_summarizer`
- **教材依据**：第8章 8.5.1 背景分析

### 第 2 步：会计分析（会计信息质量判断）

- **目标**：评估财务报表可靠性基础
- **输入**：审计报告全文
- **处理动作**：
  1. 提取审计意见类型
  2. 提取关键审计事项
  3. 识别审计机构及变更情况
  4. 识别重要会计政策和会计估计
  5. 判断会计信息可靠性等级
- **输出**：会计信息可靠性评估报告
- **工具**：`audit_opinion_reader`
- **教材依据**：第8章 8.5.2 会计分析；教材指出"可以通过关注审计意见的类型与措辞间接判断会计质量，不必开展单独的会计分析"

### 第 3 步：数据清洗与标准化

- **目标**：科目映射、单位统一、勾稽校验
- **输入**：原始结构化数据集
- **处理动作**：
  1. 科目名称标准化映射
  2. 数据单位统一
  3. 计算衍生科目（核心利润、经营资产、投资资产、有息负债等）
  4. 缺失值标记
  5. 勾稽关系校验（资产负债表平衡、四表联动）
- **输出**：标准化财务数据集 + 数据质量报告
- **工具**：`report_standardizer`, `data_validator`
- **教材依据**：第2章各类指标计算基础

### 第 4 步：财务比率计算

- **目标**：系统计算全部财务指标体系
- **输入**：标准化财务数据集
- **处理动作**：
  1. 偿债能力指标（流动比率、速动比率、现金比率、资产负债率、资产金融负债率、利息保障倍数等）
  2. 营运能力指标（各项周转率、营业周期、现金转化周期）
  3. 盈利能力指标（毛利率、核心利润率、经营资产报酬率、总资产报酬率、ROE、ROIC）
  4. 成长能力指标（各项增长率、可持续增长率）
  5. 现金流量指标（核心利润获现率、投资收益获现率、自由现金流）
  6. 上市公司特殊指标（EPS、PE、PB、每股经营现金流等）
- **输出**：完整指标数据表（各期 × 所有指标）
- **工具**：`ratio_calculator`
- **特别强调**：
  - 教材明确反对将经营现金流与净利润直接比较（口径差异大）；应使用核心利润获现率
  - 对合并财务报表和母公司财务报表分别计算两套比率
  - 偿债能力比率对合并报表不具有个体化意义（母子公司为独立法律主体）
- **教材依据**：第2章、第8章

### 第 5 步：资产结构分析与企业战略识别

- **目标**：从资产结构透视企业资源配置战略
- **输入**：标准化财务数据集（母公司报表优先）
- **处理动作**：
  1. 剔除货币资金后，将母公司资产分为经营资产和投资资产
  2. 按两类资产比重，识别企业资源配置战略类型：
     - **经营主导型**：经营资产占比>60%，专注于自身生产经营
     - **投资主导型**：投资资产占比>60%，以对外投资为主要扩张方式
     - **经营与投资并重型**：两者占比均在40%-60%之间
  3. 分析经营资产内部结构（流动资产vs非流动资产、固定资产比重等）
  4. 与年报中企业战略宣言对比，判断吻合度
- **输出**：资源配置战略识别报告
- **工具**：`structure_analyzer`, `strategy_identifier`
- **教材依据**：第3章 3.5 资产结构质量分析与企业资源配置战略的考察

### 第 6 步：项目质量分析（核心环节）

- **目标**：深入各资产、负债、利润具体项目层面分析质量
- **输入**：标准化财务数据集 + 报表附注
- **处理动作**：
  1. **资产项目质量**：货币资金（自由度/闲置率/生成质量）、应收票据与应收款项融资（信用等级）、应收账款（账龄/周转/保值）、存货（构成/物理质量/盈利/周转/保值）、长期股权投资（投资方向/收益/减值）、固定资产（成新率/产能利用率）、在建工程（进度/转固情况）、无形资产（构成/研发支出）、商誉（减值风险/业绩承诺）
  2. **负债项目质量**：流动负债强制性分析、经营负债质量（与存货和收入的匹配）、金融负债规模与成本、或有负债风险
  3. **利润项目质量**：收入质量（收现/关联交易/客户集中度）、成本费用质量（波动合理性）、核心利润持续性、非经常性损益影响
- **输出**：项目质量分析报告
- **工具**：`asset_quality_analyzer`, `earnings_quality_analyzer`
- **教材依据**：第2章项目质量分析法 + 第3-5章各项目分析

### 第 7 步：资本结构质量与资本引入战略分析

- **目标**：分析资本引入战略及资本结构质量
- **输入**：标准化财务数据集 + 所有者权益变动表
- **处理动作**：
  1. 区分四类资本来源：经营负债资本、金融负债资本、股东入资资本、留存资本
  2. 分析造血（留存+经营负债）vs 输血（金融负债+股东入资）结构
  3. 评估资本成本与投资效益的匹配性
  4. 评估资本期限结构与资产结构的协调性（防止短贷长投）
  5. 评估资本结构的财务弹性
  6. 分析控制权结构与治理结构（实际控制人判断）
  7. 从格力电器案例理解：高资产负债率≠高风险（经营负债占比大、资产金融负债率低）
- **输出**：资本结构质量分析报告
- **工具**：`structure_analyzer`
- **教材依据**：第4章 + 第8章 8.5.4

### 第 8 步：合并财务报表与差量分析

- **目标**：利用差量分析法挖掘合并报表与母公司报表之间的战略信息
- **输入**：合并财务报表 + 母公司财务报表
- **处理动作**：
  1. 计算各项目的"差量"（合并数 - 母公司数）
  2. **控制性投资规模识别**：找出母公司>合并的项目（长期股权投资、其他应收款、预付款项差额），体现向子公司提供的资金
  3. **控制性投资扩张效果**：合并总资产 - 母公司总资产 = 扩张效果
  4. **集团资金管控模式**：
     - 集权式：母公司货币资金和借款规模接近合并数
     - 分权式：母公司货币资金和借款远小于合并数
     - 混合式：兼具两者特征
  5. **母子公司业务关联度**：
     - 一致对外型：合并收入远大于母公司收入（多元化经营）
     - 母公司向子公司销售型：母公司毛利率较低、销售费用低
     - 子公司向母公司销售型：合并营业成本可能小于母公司
  6. 分析少数股东权益/损益合理性
  7. 商誉减值风险评估
- **输出**：合并报表差量分析报告
- **工具**：`consolidation_analyzer`（新工具）
- **教材依据**：第7章 + 第8章 8.3.4

### 第 9 步：趋势分析与横向比较

- **目标**：多期趋势 + 行业对标
- **输入**：多期指标数据表 + 行业数据
- **处理动作**：同比/环比/复合增长率、趋势方向识别、行业百分位排名
- **输出**：趋势分析报告 + 横向比较报告
- **工具**：`trend_analyzer`, `peer_comparator`
- **教材依据**：第2章 2.1.2 + 第8章 8.4

### 第 10 步：异常识别与风险扫描

- **目标**：基于教材风险信号自动扫描
- **输入**：全部分析结果
- **处理动作**：执行18条风险规则、按等级排序、判断持续性
- **输出**：风险扫描报告
- **工具**：`risk_scanner`
- **教材依据**：第5章"利润质量恶化的12种表现" + 各章综合风险信号

### 第 11 步：综合评价与报告生成

- **目标**：综合判断企业财务健康度，生成标准化报告
- **输入**：前10步全部分析结果
- **处理动作**：
  1. 按照"资本结构质量→资产质量→利润质量→现金流量质量"路径综合判断
  2. 评价控制性投资扩张效果
  3. 预测企业发展前景
  4. 给出综合评级（优秀/良好/一般/关注/风险）
  5. 按模板生成报告
- **输出**：完整财务报表分析报告
- **工具**：`comprehensive_evaluator`, `report_generator`
- **教材依据**：第8章 8.3.5 财务状况质量整体评价 + 8.3.6 企业发展前景预测

## 五、关键概念速查

### 5.1 教材独创/改进指标

| 指标 | 公式 | 设计目的 |
|------|------|---------|
| 核心利润 | 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 利息费用 | 替代"大杂烩"的营业利润 |
| 核心利润率 | 核心利润/营业收入 | 反映主业真实盈利能力 |
| 核心利润获现率 | 经营活动现金净流量/核心利润 | 评判利润含金量（传统工业1.2-1.5理想） |
| 资产金融负债率 | 金融性负债/总资产 | 替代传统资产负债率 |
| 应收款项周转率 | 营业收入/(应收票据+应收账款+应收款项融资) | 弥补应收账款周转率在使用商业汇票时的缺陷 |
| 经营资产周转率 | 营业收入/经营资产平均余额 | 排除投资资产干扰 |
| 经营资产报酬率 | 核心利润/经营资产平均余额 | 衡量经营资产真实回报 |
| 投资收益获现率 | 取得投资收益收到的现金/投资收益 | 衡量投资收益现金能力 |

### 5.3 资产质量三维度

教材从三个维度评价各资产项目质量：
- **盈利性**：资产能否带来经济利益（核心指标：毛利率、核心利润率、经营资产报酬率）
- **周转性**：资产能否及时变现或消耗（核心指标：各项周转率）
- **保值性**：资产是否面临减值风险（关注减值准备计提充分性）
